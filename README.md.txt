Speech Emotion Recognition using Wav2Vec 2.0

A GUI-based system to detect emotions from speech using Wav2Vec 2.0 feature extraction and Random Forest classification.  
Built with Streamlit and trained on SAVEE + RAVDESS datasets.

Tech Stack
Python, Streamlit, scikit-learn, SHAP, librosa, Wav2Vec 2.0

Output
- Emotion prediction with probability
- SHAP visual explanation
- Audio waveform display

How to Run
Anaconda bash
pip install -r requirements.txt
streamlit run app.py
